Youtube


Fecha de lanzamiento:

febrero de 2005

Desarrollado por:

Steve Chen, Chad Hurley y Jawed Karim

Empresa:

YouTube, LLC (adquirida por Google)

Descripción:

YouTube utiliza algoritmos avanzados de recomendación basados en el historial de visualización y las interacciones de los usuarios para sugerir videos que podrían ser de su interés, lo que mantiene a los usuarios comprometidos con la plataforma.

Tecnologías/herramientas implementadas:

  • Python
  • Javascript
  • HTML5
  • Google Cloud Platform
  • TensorFlow

Instagram

Fecha de lanzamiento:

octubre de 2010

Desarrollado por:

Kevin Systrom y Mike Krieger

Empresa:

Instagram, Inc. (ahora Meta Platforms, Inc.)

Descripción:

Instagram revolucionó la fotografía móvil con su amplia gama de filtros y herramientas de edición en tiempo real, permitiendo a los usuarios mejorar y personalizar sus fotos antes de compartirlas.

Tecnologías/herramientas implementadas:

  • Python
  • React Native
  • Javascript
  • Django
  • PostgreSQL

Facebook

Fecha de lanzamiento:

febrero de 2004

Desarrollado por:

Mark Zuckerberg, Eduardo Saverin, Andrew McCollum, Dustin Moskovitz y Chris Hughes.

Empresa:

Facebook, Inc. (ahora Meta Platforms, Inc.)

Descripción:

Facebook ofrece un feed de noticias altamente personalizado utilizando algoritmos que consideran las interacciones de los usuarios, sus amigos y sus preferencias para mostrar contenido relevante y mantener a los usuarios enganchados.

Tecnologías/herramientas implementadas:

  • PHP
  • React/li>
  • MySQL
  • GraphQL
  • Cassandra

Twitter

Fecha de lanzamiento:

marzo de 2006

Desarrollado por:

Jack Dorsey, Noah Glass, Biz Stone y Evan Williams

Empresa:

Twitter, Inc.

Descripción:

Twitter permite la publicación y visualización de actualizaciones en tiempo real, facilitando la comunicación instantánea y la difusión de noticias, eventos y tendencias a nivel global.

Tecnologías/herramientas implementadas:

  • Javascript
  • MySQL
  • Scala
  • Redis
  • Ruby on Rails

Whatsapp

Fecha de lanzamiento:

noviembre de 2009

Desarrollado por:

Jan Koum y Brian Acton

Empresa:

WhatsApp Inc. (ahora parte de Meta Platforms, Inc.)

Descripción:

WhatsApp ofrece mensajería instantánea con cifrado de extremo a extremo, asegurando que solo los remitentes y los destinatarios puedan leer los mensajes intercambiados.

Tecnologías/herramientas implementadas:

  • Erlang
  • Java
  • Swift
  • WebRTC
  • AWS
